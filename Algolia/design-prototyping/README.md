# Design Prototyping

Interactive prototypes for design exploration and engineering handoff. Each folder is a standalone project.

## Prototypes

| Folder | Description | Stack |
|--------|-------------|-------|
| [`agent-studio/`](./agent-studio) | Agent Studio UI prototype | React, Vite, Tailwind |
| [`mcp-flows/`](./mcp-flows) | MCP Flows builder prototype | React, Vite, Tailwind |

## Getting started

Each prototype is self-contained. To run one locally:

```bash
cd <prototype-folder>
npm install
npm run dev
```
